Plongée Plus Profonde dans la Structure du HTML

Nous avons maintenant une idée du fonctionnement des documents HTML.

Chaque document HTML suit une structure standard, définie par des balises spécifiques. Ces balises aident les navigateurs à comprendre et afficher correctement le contenu.

Récapitulatif

  • html : L'élément racine d'un document HTML.
  • head : Contient les métadonnées du document, comme son titre ( title ) et les liens vers des styles ou des scripts.
  • body : Contient tout le contenu qui sera affiché dans le navigateur.

Il est maintenant temps d'explorer davantage la structure.

Créer des Listes

HTML vous permet de créer deux types de listes : ordonnées et non ordonnées. Les deux utilisent la balise li pour les éléments individuels de la liste.

Listes Ordonnées

Les listes ordonnées ol affichent les éléments avec des numéros.

Listes Non Ordonnées

Les listes non ordonnées ul affichent les éléments avec des points.
